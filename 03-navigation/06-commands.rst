===============
Useful commands
===============

These commands support both absolute and relative paths

-------

**mkdir <directory_name>**:
    create directory
**rm <delete_me>**: 
    remove file / directory
**mv <original_location> <new_location>**:
    move file / directory
**cp <original> <copy>**:
    copy file / directory

-------

Work in "workspace" directory:
    - **cd workspace**
    - **pwd** to confirm

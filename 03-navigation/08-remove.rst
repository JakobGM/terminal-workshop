=============================
rm : remove file or directory
=============================

Syntax: **rm [-r] <dir/file>**


- Remove file:
    **rm delete_me**

- Remove directory:
    **rm another_sibling**

- Once again...
    **rm -r another_sibling**

- Why the difference?
    **-r** : *Recursive*, want to delete the directory and its sub directories / files
